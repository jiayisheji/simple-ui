何时使用

按钮组遵循正常的按钮设计准则。 有关类别和大小的所有 Button 选项都适用于 Button Groups 。 以下所有功能均可提供标准尺寸和小尺寸的填充，轮廓和文字。

多个按钮组

Checkbox

当可以从中选择一小部分选项时使用，类似于 Checkbox 组件

  • 默认外观填充表示已选择一个选项，轮廓表示未选择的选项
  • 用户可以单击按钮进行选择，然后再次单击以取消选择
  • 可以配置预选选项
format_align_left format_align_center format_align_right format_align_justify

Radio

当从一个小的选项列表中选择一个选项时使用，类似于 Radio 组件

  • 默认外观填充表示已选择一个选项，轮廓表示未选择的选项
  • 用户可以单击按钮进行选择，然后再次单击同样的选项无法取消选择
  • 可以配置预选选项
