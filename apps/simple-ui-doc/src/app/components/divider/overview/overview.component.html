何时使用

  • 分隔线将内容分成清晰的组。
  • 对不同章节的文本段落进行分割。
  • 对行内文字/链接进行分割，例如表格的操作列。

原则

精细的

分隔线应该在布局中很明显，但不突兀。

次要的

只有当元素不能使用空格分隔时，才应该使用分隔线。

稀少的

谨慎使用分隔线，以创建分组而不是独立作为项目。
